# FocusNOW

FocusNOW adalah aplikasi berbasis **Next.js** yang dirancang untuk membantu produktivitas belajar dengan menyediakan **timer belajar** pada halaman utama serta halaman terpisah untuk **manajemen tugas** yang dapat ditambahkan dan dihapus oleh pengguna.
